Características del Sistema

Extracción Automática

Sube un PDF y el sistema extrae automáticamente título, autores, año, DOI y más metadatos.

Filtrado Avanzado

Filtra artículos por año, estado, LGAC, autor y más criterios para encontrar lo que necesitas.

Exportación Excel

Genera archivos Excel con el formato institucional requerido por el Cuerpo Académico.

Inicio Rápido

  1. Sube un PDF de un artículo usando el botón "Subir PDF"
  2. Revisa los metadatos extraídos automáticamente
  3. Completa la información faltante en el formulario
  4. Guarda el artículo en la base de datos
  5. Exporta a Excel cuando necesites el reporte institucional
